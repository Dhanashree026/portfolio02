# Portfolio Website

This repository contains the source code for my personal portfolio website.

The website is built using core web technologies and follows a multi-page structure. 
Layout and styling are implemented using HTML and CSS, while JavaScript is used to 
add interactivity and smooth user experience.

## Technologies

HTML5  
Used to structure the pages and define the layout.

CSS3  
Used for styling, layout design, color scheme, and responsiveness.

JavaScript (script.js)  
Used for interactive elements such as smooth scrolling, hover effects, 
and dynamic user interface behavior.

## Project Structure

about.html  
About Me page containing personal information, background, and contact details.

resume.html  
Resume section showing education, certifications, technical skills, and soft skills.

projects.html
Projects section contain projects and their informations.

other.html  
Page containing projects, interests, languages, and hobbies.

css  
style.css  
Contains all the styling of the website including layout, colors, fonts, responsiveness, and visual design.

js  
script.js  
Handles interactive functionality such as smooth scrolling, hover effects, and dynamic UI behavior.

images  
Stores all images used in the portfolio such as profile pictures and design assets.

README.md  
Documentation file describing the project, technologies used, and structure of the repository.
